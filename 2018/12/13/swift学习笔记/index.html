Swift学习笔记

2018-12-13

面向过程编程

  • 定义常量 、变量
    let 常量名
    var 变量
  • 常量赋值
    let 常量名 = 值
    （注：常量必须进行初始化，否则会报错）

  • 基本数据类型
    整数：Int
    浮点数：Double，Float
    字符：Character
    字符串：String
    布尔类型：Bool（只有true和false）
    元祖类型：Tuple（定义变量的一个组合）

  • 可选类型Optional：代表变量可能有值的情况
    var 变量 ：类型？,默认是无值（nill）

  • 集合类型
    定义：Array<类型>或[类型]
    有默认值的数组，array = [ Int ] (repeatElement( ,count: ))
    有序范围的Int数组，Array（起始值…终止值）
    insert 插入
    remove 移除
    contains 检查是否包含
    sorted 转换为数组
  • 控制流
    if else
    switch(超过两个以上条件判断的场合)和case合用(一个变量多个条件判断)默认default：（表示不在switch规定的范围内）
    for…in…
    while
  • 字典
    Dictionary <键类型，值类型>或[键类型：值类型]
    计数 count，空否：is Empty
    添加或更新 字典变量 [键] = 值
    循环一个字典
    for （key,value）in 字典变量{
      print(key,value)
}
    把键值对分离成数组，用[数组类型] （字典变量，keys），[数组变量]（字典变量.values）
  • 闭包（一种函数的简写形式，省去函数名，把参数和返回值放入花括号内）
    捕获和存储定义在其上下文中的任何常量和变量的引用
    {(parameters/接收的参数) -> (return type/闭包返回值类型) in
    stasements/保存在闭包中需要执行的代码
    }
  • 枚举
    enum

    面向对象编程

  • 类 class
    getter 描述属性如何计算及返回语句，get{语句和返回}
    setter 有新值后如何影响其他属性,set{语句}
    class 类名：SuperClass{
    定义属性和方法
    }
    （注：定义的类可以没有父类，那么该类是rootClass
    通常情况下，定义类时，继承自NSObject（非OC的NSObject））

    • 类的属性
      存储属性：实类型的一部分，用于存储常量和变量
      计算属性：通过某种方式计算出来的属性
      类属性：与整个类自身相关的属性
    • 实例化
      var shape = Shape()

  • 结构体 struct
