Markdown 学习笔记

2018-11-29

一、简介

Markdown是一种可以使用普通文本编辑器写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。其语法简洁明了，学习容易，而且功能比纯文本强，适合用来写博客。所编写的文档以“.md”的文件名保存在软件的目录下面。

二、基本语法

1.标题

# 这是一级标题
## 这是二级标题

行首可以插入六个#号，对应标题有1~6共六级，从标题大小就能分别出标题的重要性情况。
注：#后需加空格号才能形成标题

2.文字格式

  • 这是斜体字 

    *斜体字*

  • 这是粗体字

    **粗体字**

  • 这是粗斜体字

    ***粗斜体字***

  • 这是删除线 

    ~~删除线~~
