Git 学习笔记

2018-11-30

Git是目前世界上最先进的分布式版本控制系统。

基本语法

1.安装Hexo

右键选择Git Bash Here进入来执行
在命令行中输入：

  • npm install hexo-cli -g

    2.创建一个版本库:

    $ git init

    2.查看仓库的状态

    $ git status

    3.生成SSH keys

    在命令行中使用如下命令输入自己的邮箱地址
    ssh-keygen -t rsa -C “xxxx@qq.com“

    4.测试是否添加成功

    输入下面的命令，看看设置是否成功
    ssh -T git@github.com

    5.做全局的配置

    输入自己的用户名和邮箱
    $ git config –global user.name “Lu-luck”
    $ git config –global user.email “xxxx@qq.com“
