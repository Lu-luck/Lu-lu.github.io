Git 学习笔记

2018-11-30

Git是目前世界上最先进的分布式版本控制系统。

基本语法

1.安装Hexo

右键选择Git Bash Here进入来执行
在命令行中输入：

  • npm install hexo-cli -g

    2.创建一个版本库:

    $ git init

    2.查看仓库的状态

    $ git status

    3.生成SSH keys

    在命令行中使用如下命令输入自己的邮箱地址
    ssh-keygen -t rsa -C “xxxx@qq.com“

    4.测试是否添加成功

    输入下面的命令，看看设置是否成功
    ssh -T git@github.com

    5.做全局的配置

    输入自己的用户名和邮箱
    $ git config –global user.name “Lu-luck”
    $ git config –global user.email “xxxx@qq.com“

Markdown 学习笔记

2018-11-29

一、简介

Markdown是一种可以使用普通文本编辑器写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。其语法简洁明了，学习容易，而且功能比纯文本强，适合用来写博客。所编写的文档以“.md”的文件名保存在软件的目录下面。

二、基本语法

1.标题

# 这是一级标题
## 这是二级标题

行首可以插入六个#号，对应标题有1~6共六级，从标题大小就能分别出标题的重要性情况。
注：#后需加空格号才能形成标题

2.文字格式

  • 这是斜体字 

    *斜体字*

  • 这是粗体字

    **粗体字**

  • 这是粗斜体字

    ***粗斜体字***

  • 这是删除线 

    ~~删除线~~

3.插入链接、图片、音频、视频
